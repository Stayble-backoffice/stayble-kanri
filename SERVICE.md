# SERVICE.md

「乗り換え民泊管理」のサービス定義をまとめます。  
Codexは、サービス内容・やること・やらないことをこのファイルに従って表現してください。

---

## 1. サービス名

```txt
乗り換え民泊管理
```

---

## 2. サービス定義

「乗り換え民泊管理」は、すでに開業済みの住宅宿泊事業者向けに、既存管理会社からの乗り換え先として提供する低コスト型の民泊管理サービスです。

開業支援や申請代行を主目的にせず、運用中の民泊に必要な管理・清掃・リネン・OTA運用・現地対応を、Staybleの現場基盤で引き受けます。

---

## 3. 主ターゲット

対象:

- すでに住宅宿泊事業の届出済み
- すでに民泊として運用中
- 既存管理会社に不満がある
- 管理費を下げたい
- 清掃品質に不満がある
- リネン・清掃・現地対応だけちゃんとしてほしい
- 既存管理会社からの乗り換えを検討している

対象外または優先度低:

- これから民泊を始めたい人
- 物件購入前の相談
- 申請代行だけを依頼したい人
- 物件選定や内装から丸投げしたい人
- 行政書士業務を求める人

---

## 4. 主対象の制度

基本は住宅宿泊事業法に基づく届出住宅。  
旅館業・簡易宿所は初期LPの主軸にしない。

LP表現:

```txt
住宅宿泊事業法に基づく開業済み民泊を主対象としています。旅館業・簡易宿所については個別にご相談ください。
```

「住宅宿泊管理業ライセンスで旅館業も管理できます」のような表現は禁止。

---

## 5. 標準対応する業務

### 5.1 法定管理業務

登録番号発行後、住宅宿泊管理業務として以下を標準対応する想定。

- 衛生管理
- 安全管理
- 外国人宿泊者向け案内
- 宿泊者名簿の作成・保管
- 周辺環境への悪影響防止に関する説明
- 苦情・問い合わせ対応
- 定期報告
- 必要に応じた現地対応

法的な詳細は `LEGAL_REFERENCES.md` を参照。  
LPでは「完全対応」「全部代行」などの断定表現を避ける。

### 5.2 清掃

清掃はStaybleセット契約。  
既存清掃会社の継続は不可。

理由:

- 清掃品質まで責任を持つため
- 報告・写真・鍵・リネン導線を統一するため
- 外注清掃会社任せの管理会社と差別化するため

LP表現:

```txt
清掃品質まで責任を持つため、清掃・リネンはStaybleセット契約です。
```

### 5.3 リネン

リネンは標準かつ必須。  
物件特性に応じて都度集配型・ストック型を使い分ける。

### 5.4 消耗品

消耗品サービスはオプション。  
希望するオーナーのみ利用。

### 5.5 OTA運用

Stayble側でOTAを新規立ち上げる。  
既存OTAアカウントの引き継ぎは不可。  
オーナーは補助ホストになる。

標準対応:

- OTA初期立ち上げ
- OTA掲載内容の作成/整備
- 写真撮影
- 価格調整
- メッセージ対応
- レビュー返信
- 予約変更対応
- キャンセル対応
- 返金交渉
- 掲載内容修正
- カレンダー調整

### 5.6 簡易的な価格調整

「収益最大化」まで強く言い切らない。  
簡易的な価格調整として扱う。

LP表現:

```txt
稼働状況を見ながら、簡易的な価格調整にも対応します。
```

### 5.7 写真撮影

写真撮影は標準対応・無料。  
ただしプロカメラマン撮影ではなく、Stayble側の標準撮影。

LP注釈:

```txt
※写真撮影はStayble標準撮影です。プロカメラマンによる撮影や大規模な撮影ディレクションは含みません。
```

### 5.8 収益シミュレーション

標準対応・無料。  
ただし売上・稼働率・利益を保証しない。

## 6. やらない業務

以下は原則やらない。

- 申請代行
- 行政書士業務
- 許認可コンサル
- 物件選定
- 不動産購入サポート
- 内装工事
- 家具家電コーディネートの本格代行
- 消防設備工事そのもの
- 宿泊税/税務申告代行
- 旅館業許可申請
- 宿泊施設の完全な収益保証

---

## 7. 提携先紹介なら可能な業務

- 行政書士
- 消防設備関連
- 内装/修繕業者
- 鍵/スマートロック業者
- その他専門業者

LP表現:

```txt
申請代行は行っていません。必要に応じて専門家のご紹介は可能です。
```

---

## 8. 乗り換え時の無料対応

以下は標準対応・無料。

- 初期費用
- 写真撮影
- OTA初期立ち上げ
- 簡易収益シミュレーション
- 既存運用のヒアリング
- PMS/OTA運用設計
- 清掃・リネン導線の設計

LPで強く打ち出してよい。

## 10. サービスの差別化

通常の管理会社:

```txt
管理会社 → 清掃会社 → リネン会社 → 現地対応者
```

Stayble:

```txt
清掃・リネン・現場報告を持つStaybleが、管理業務を上に乗せる
```

LPでは以下の表現が強い。

- 清掃会社が管理を請け負う時代へ
- 清掃品質まで責任を持つ管理会社
- 管理費を抑えながら、現場は薄くしない
- 余計なフル代行ではなく、必要な管理だけ
