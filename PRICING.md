# PRICING.md

「乗り換え民泊管理」の料金表現・料金表・手残りシミュレーション・注釈をまとめます。  
Codexは料金まわりのコピーやUIを作る際、必ずこのファイルに従ってください。

---

## 1. 最重要ルール

```txt
管理費 ＝ 月額固定費5,000円〜 + 売上歩合5%
```

固定月額と売上歩合をまとめて「管理費」と呼ぶ。  
絶対に「管理費5%」と書かない。

---

## 2. 正しい料金コピー

OK:

```txt
月額5,000円〜 + 売上5%
```

```txt
管理費は、物件規模別の月額固定費と売上歩合5%で構成されます。
```

```txt
管理費（月額固定費 + 売上5%）
```

NG:

```txt
管理費5%
```

```txt
5%の管理費
```

```txt
管理費は売上の5%のみ
```

## 5. 必須注釈

料金表示の近くに必ず入れる。

```txt
※月額5,000円〜は管理費の月額固定部分です。管理費は、物件規模別の月額固定費と売上歩合5%で構成されます。
※清掃費・リネン費・消耗品費・ゴミ処理関連費・駆け付け費・その他実費は別途発生します。
※物件規模・所在地・運用内容により、個別見積もりとなる場合があります。
```

写真/OTA無料訴求の近くには以下。

```txt
※写真撮影はStayble標準撮影です。プロカメラマン撮影や大規模な撮影ディレクションは含みません。
※OTA初期立ち上げは通常の乗り換え導入に必要な範囲での対応です。特殊な販売戦略や大幅な掲載改善は個別相談となる場合があります。
```

## 6. 費用構造

LPでは以下の構造を明示する。

```txt
管理費
+ 清掃費
+ リネン費
+ 消耗品費（任意）
+ ゴミ処理関連費
+ 駆け付け費
+ その他実費
```

管理費以外は、物件条件や運用内容により変動する。

## 8. 駆け付け費

LPでは具体金額を出さない。  
表現は「固定料金制」とする。

LP表現:

```txt
駆け付け対応は固定料金制。対応内容は後日オーナー様へ報告します。
```

契約前案内で出す内容:

- 固定料金
- 移動費実費
- 22時以降は深夜割増25%
- 作業時間は1回1時間まで
- 超過料金なし
- 解決/未解決にかかわらず後日報告

## 9. 手残りシミュレーション

LPでは具体例を出す。  
添付スケッチのように、積み上げ棒グラフで視認しやすくする。

### 比較UIの考え方

左: 一般的な管理会社  
右: Stayble

積み上げ要素:

- オーナー手取り
- 清掃費
- 管理費
- その他費用がある場合は注釈で扱う

Stayble側の管理費は必ず以下のラベルにする。

```txt
管理費（月額固定 + 売上5%）
```

### 比較例

例: 月間売上300,000円、規模Aの場合

一般的な管理会社の例:

```txt
管理費率20%の場合：60,000円
```

Stayble:

```txt
管理費：20,000円（月額5,000円 + 売上5% 15,000円）
```

差額:

```txt
管理費だけで月40,000円の差
```

この比較は「管理費部分の比較」として扱う。

### シミュレーション必須注釈

```txt
※上記は一定条件に基づく概算シミュレーションです。売上・稼働率・手取り額を保証するものではありません。
※比較対象の管理費率は一例であり、すべての管理会社に該当するものではありません。
※清掃費・リネン費・消耗品費・ゴミ処理関連費・駆け付け費・その他実費は物件条件や運用内容により変動します。
※実際の手残りは、売上、稼働率、清掃回数、リネン方式、消耗品利用有無、設備トラブル等により変動します。
```

## 10. 初期費用

LPで出す。

```txt
初期費用0円
写真撮影0円
OTA初期立ち上げ0円
```

補足:

- 撮影はStayble標準撮影
- OTAはStayble側で新規立ち上げ
- 既存OTA引き継ぎ不可
- オーナーは補助ホストになる

## 11. 契約関連の費用

違約金などはLPに出さない。  
`CONTRACT.md` にのみ記載する。

LPに記載禁止:

- 半年以内の違約金
- 半年以降の違約金
- 残月数分
- 契約解除条件の詳細
